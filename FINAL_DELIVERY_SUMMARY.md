## 🎓 HOW TO USE

### Download:
1. Click menu (top right in v0)
2. Select "Download ZIP"
3. Extract the folder

### Install:
```bash
cd virtual-learning-academy
pnpm install
```

### Test:
```bash
pnpm dev
# Open http://localhost:3000
# Test on different screen sizes
```

### Deploy:
1. Push to GitHub
2. Connect to Vercel
3. Deploy with 1 click
4. Share live URL

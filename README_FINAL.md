## 🚀 HOW TO USE

### 1. Download
- Click menu (top right in v0)
- Select "Download ZIP"
- Extract the folder

### 2. Install Locally
```bash
cd virtual-learning-academy
pnpm install
```

### 3. Test Locally
```bash
pnpm dev
```
Open http://localhost:3000 and test on:
- Desktop browser (right-click → Inspect → Toggle device toolbar)
- Mobile devices (if available)
- Different screen sizes

### 4. Deploy to Vercel
```bash
# Push to GitHub first
git add .
git commit -m "Initial commit"
git push

# Then connect to Vercel and deploy
```

### 5. Share Your Website
- Share the live Vercel URL with your audience
- Use it in marketing materials
- Share on social media

## 🎓 CUSTOMIZATION GUIDE

### Change Colors
Edit `globals.css` or `tailwind.config.js`:
```css
--maroon: #8C1B2E;
--maroon-light: #B43A4E;
```

### Update Content
Edit each page's `page.tsx` file:
- Update text, images, and links
- Modify program details
- Add your information

### Add Pages
Create new files in `/app`:
```bash
mkdir app/new-page
touch app/new-page/page.tsx
```

### Deploy Changes
```bash
git add .
git commit -m "Your message"
git push
# Vercel auto-deploys
```
